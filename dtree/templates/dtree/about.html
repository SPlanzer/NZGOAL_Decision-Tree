NZGOAL Decision Tree

Prior to loading a dataset the LINZ Data Service (LDS) it must be confirmed the NZGOAL Decision Tree has been followed and adhered to.

By navigating to the "New Data Set" page a new dataset can be evaluated prior to publishing and shall only be published once it has been proven to meet the NZGOAL Framework Decision Tree.

Auditing

Once a dataset has been published the LDS id must be added against the dataset (navigate to "Datasets") This id is used to perform the audit function of this website (via the "Audit" page). The audit comapres all new LDS dataset ids for a set period to the ids as stored as part of this website.
