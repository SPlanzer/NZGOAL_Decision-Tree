NZGOAL decision tree

Prior to loading a dataset the LINZ Data Service it must be confirmed the NZGOAL Decision Tree has been followed and adhered to

By navigating to the "add dataset" page a new dataset can be evaluated prior to publishing and shall only be published once it has been proven to meet the NZGOAL frame work decision tree.

Once a dataset has been published the LDS id must be added against the dataset (navigate to "datasets") This id is used to perform the audit function. At determined times via the "Audit" audits are ran. This all new data set ids between audit dates are comapred to the datasets stored as part of this web interface to comfirm that they have gone through the NZGOAL decsion tree prior to publishing
